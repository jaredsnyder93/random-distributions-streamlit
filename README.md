This repository contains a simple Streamlit application demonstrating the
Central Limit Theorem.

The app shows how the distribution of sample means converges to a normal
distribution as sample size increases, even when the underlying data come
from non-normal distributions such as the exponential distribution.

## Live App
Deployable on:
https://share.streamlit.io/

## What the App Demonstrates
- Convergence of sample means to a normal distribution
- Effect of sample size on variance
- Intuition behind the Central Limit Theorem

## Technologies Used
- Python
- Streamlit
- NumPy
- Matplotlib
- SciPy
